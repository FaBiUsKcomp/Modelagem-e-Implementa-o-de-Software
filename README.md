*Abaixo está um exemplo da sequência de **Fibonacci** escrita em Javascript:*

```
const number = 5

let n1 = 0, n2 = 1, nextTerm;

for (let i = 1; i <= number; i++) {
    console.log(n1);
    nextTerm = n1 + n2;
    n1 = n2;
    n2 = nextTerm;
}
```
